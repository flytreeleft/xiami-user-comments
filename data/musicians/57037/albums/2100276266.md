## 简介

<div>
<p>广州 05月21号 乐府 《怎么搞的？》与雷磊 <a data-spm-anchor-id="a1z1s.6632057.389823137.5" href="https://www.douban.com/event/26619388/?spm=a1z1s.6632057.389823137.5.i9gATj" target="_blank" rel="nofollow"></a><a href="https://www.douban.com/event/26619388/" target="_blank" rel="nofollow noreferrer noopener">https://www.douban.com/event/26619388/</a><span style="font-family: arial, sans-serif; font-size: 12.8px; line-height: normal;"><br><br>
</span></p><br>
<p><span style="font-family: arial, sans-serif; font-size: 12.8px; line-height: normal;">《What's up with That？》是<span style="font-family: arial, sans-serif; font-size: 12.8px; line-height: normal;">电子音乐人Dave Liang（</span><span style="font-family: arial, sans-serif; font-size: 12.8px; line-height: normal;">上海复兴方案）和</span>视觉艺术家雷磊</span><span style="font-family: arial, sans-serif; font-size: 12.8px; line-height: normal;">合作的多媒体艺术项目。</span><span style="font-family: arial, sans-serif; font-size: 12.8px; line-height: normal;">他们用现场电子音乐演奏和视觉vj表演的方式，</span><span style="font-family: arial, sans-serif; font-size: 12.8px; line-height: normal;">描绘孩子们与生俱来的好奇心，和天马行空的想像力。</span><span style="font-family: arial, sans-serif; font-size: 12.8px; line-height: normal;">10首电子音乐的创作从雷磊孩童时期讲故事的磁带录音出发，</span><span style="font-family: arial, sans-serif; font-size: 12.8px; line-height: normal;">在其基础上Dave进行了采样和编辑，</span><span style="font-family: arial, sans-serif; font-size: 12.8px; line-height: normal;">并加入电子音乐的元素和当代城市生活的声音。</span><span style="font-family: arial, sans-serif; font-size: 12.8px; line-height: normal;">视觉方面的动画创作，来自于雷磊对父亲80年代旧书《</span><span style="font-family: arial, sans-serif; font-size: 12.8px; line-height: normal;">国际书籍装帧设计选》的再创作，</span><span style="font-family: arial, sans-serif; font-size: 12.8px; line-height: normal;">用手工拼贴的方式制作了一系列充满了几何形体和抽象设计感的动画 。</span></p><br>
<p> </p><br>
